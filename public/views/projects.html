Arduino-Powered Costume Gun

Technologies: C++

For part of a costume for the Nine Worlds convention this year, I commissioned a 3D-printed model of Fiona's pistol from Tales from the Borderlands. It includes 12 LEDs, a trigger button, and reed switches to detect the barrel orientation, which are all linked to an Arduino micro, which I am programming to have a wide range of behaviours.

GitHub Project: https://github.com/braiba/tftb-fiona-pistol-arduino

Live Text Adventures

Technologies: PHP, MySQL, JS, CSS

I have organised Live Text Adventure events based on a concept from https://en.wikipedia.org/wiki/GameCity#GameCity_Festivals in 2014. It involves a number of writers simulating a game in the style of classic text adventure games against a series of players. Each writer runs the game for a player simultaneously, with all the games being displayed on a big screen that it visible to everyone who is not currently playing. This means as new players start to play the game in future rounds they have the combined knowledge of all previous players and can therefore progress further.

GitHub Project: https://github.com/braiba/livetextadventurest
