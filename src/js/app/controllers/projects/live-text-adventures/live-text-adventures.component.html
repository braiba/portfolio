Live Text Adventures

Technologies: PHP MySQL JS CSS

I have organised Live Text Adventure events based on a concept from GameCity in 2014. It involves a number of writers simulating a game in the style of classic text adventure games against a series of players. Each writer runs the game for a player simultaneously, with all the games being displayed on a big screen that it visible to everyone who is not currently playing. This means as new players start to play the game in future rounds they have the combined knowledge of all previous players and can therefore progress further.
